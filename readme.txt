Allumer la machine virtuelle
Aller dans le dossier système de fichier/home
Ouvrir le Terminal à cette endroit et y rentrer la commande "sudo mkdir site"
Cette action va créer un dossier site, sudo étant pour avoir les droits admin
Ouvrir le terminal dans le dossier créer et rentrer la commande "sudo touch README"
Ceci créer un ficher appelé README.txt
Pour installer Jekyll commande "sudo apt-get install Jekyll"
Pour installer Ruby "sudo apt-get install Ruby-full"
Pour faire des mises à jour "gem update Jekyll"
Sudo Jekyll New "monblog"
Lancer un serveur "sudo jekyll serve"
